Works on any Device

Built with Bootstrap, the world’s most popular front-end framework, the theme powers a responsive, mobile-first layout that automatically adapts to any device your visitors may be viewing your website on.

And with the theme’s perfect integration with Bootstrap’s multicolumn grid system, you can present desktop visitors your content in any shape, that will gracefully downgrade on mobile devices. Additionally, you can showcase any type of media formats in a Pinterest-style grid powered by the Masonry script, that repositions itself automatically when the device changes its width .

Optimized For Speed

Page load times are not only a ranking factor, but also an integral part of any visitor’s experience. Website speed should be a priority for every web publisher.

The theme uses website speed best-practices like prioritizing content by placing scripts in the page’s footer, using icon fonts and avoiding decorative background images.

No-Nonsense Features

Animated Content, Video Backgrounds, Content Shortcodes, Lazy Loading Images. Yeah, we don’t have those.

And if you think they’re cool you’d better reconsider. Studies have shown that pages with things moving around and making sounds outside visitors’ control are driving them away.
